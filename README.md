This is an [Express.js](https://expressjs.com/) and [Socket.io](https://socket.io/docs/v4/) project

## Quick Guide

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3500](http://localhost:3500) with your browser to see the result.
